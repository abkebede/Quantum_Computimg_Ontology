# Advanced Codes Expansion Update

This is **not a full replacement** for `qc-code-study-library`. It is a partial update package that adds intermediate and advanced code examples requested for the library.

## New code families added

- Simple QAOA for MaxCut
- Simulator vs noisy-simulator comparison
- IBM hardware workflow comparison template
- Transverse-field Ising spin-chain Hamiltonian
- Heisenberg spin-chain Hamiltonian
- Two-site Hubbard model Hamiltonian
- Quantum phase estimation introduction
- Amplitude estimation introduction
- Quantum machine-learning kernel introduction
- Error-mitigation / readout-mitigation introduction
- Expanded quantum chemistry Hamiltonian / VQE example
- OpenQASM runner notes

## Important note

These files are designed as teaching and study codes. Some advanced examples are simulator-first. Hardware execution examples are token-safe templates and must not include a real IBM Quantum API key in GitHub.

Suggested commit message:

```text
Add intermediate and advanced quantum computing code expansion
```
