# Basic Measurement Teaching Unit Update

This is not a full replacement package. It is a partial update for the existing `qc-code-study-library` repository.

- `10_application_notes/beginner/one_qubit_measurement_counts_application_note.md`
- `ontology/examples/one_qubit_measurement_counts_metadata.yaml`
- `ontology/code_index_basic_measurement_addition.csv`

## Purpose

This update documents the first beginner measurement pathway:

```text
qubit initialization -> X gate -> measurement -> shots -> counts -> interpretation
```

This unit should be used before Hadamard superposition because it teaches the difference between a deterministic measurement and a probabilistic measurement.
