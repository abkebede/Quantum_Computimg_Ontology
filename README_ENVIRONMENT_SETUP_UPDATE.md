# Environment and Jupyter Workflow Teaching Unit Update

This is **not a full replacement** of the repository. It is a partial update for the existing project folder named `qc-code-study-library`.

## Added teaching unit

This update adds a beginner teaching unit for the reproducible setup workflow:

- creating and activating `qgss_env`
- installing Qiskit and support packages
- verifying the Python executable
- distinguishing terminal commands from Python code
- launching Jupyter Notebook from the project folder
- running `.py` files directly
- using `.ipynb` notebooks for line-by-line study
