# Beginner Learning Path

1. Create and verify the Qiskit environment.
2. Run `environment_check.py`.
3. Run a one-qubit X gate.
4. Run a Hadamard superposition code.
5. Study measurement counts and shots.
6. Run a Bell-state circuit.
7. Modify one line at a time and record what changes.
8. Study Sampler and Estimator primitives.
9. Study parameterized circuits.
10. Use IBM hardware only after local simulator examples are understood.
11. Move to Grover, QFT, VQE, and OpenQASM examples.

For every code, ask:

- What problem does this code represent?
- What are the important gates?
- What output should I expect?
- What line can I modify safely?
- What real or natural problem is connected to this code?
