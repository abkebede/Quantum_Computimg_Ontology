Quantum Computing Code Ontology
The ontology organizes this repository by concepts, code type, algorithms, mathematics, applications, and learner level. It helps beginners and novice users answer: "What should I study next?" and "What does this code actually represent?"
Core files:
`qc_code_ontology.md` — conceptual ontology for the library.
`qc_code_metadata_schema.yaml` — metadata template for each code example.
`code_index.csv` — searchable table of examples.
`concept_index.md` — concept-to-code map.
`algorithm_index.md` — algorithm-to-code map.
`application_index.md` — application-to-code map.
`beginner_learning_path.md` — recommended order for beginners.
`contribution_template.md` — template for adding new examples.
`examples/` — sample metadata cards.
