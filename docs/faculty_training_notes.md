# Faculty Training Notes

This repository can support a short faculty workshop in basic quantum computing.

## Workshop sequence

1. Explain the environment: Python, qgss_env, Qiskit, Jupyter.
2. Run `environment_check.py`.
3. Run a one-qubit X gate example.
4. Run a Hadamard superposition example.
5. Run a Bell-state example.
6. Explain Sampler and Estimator.
7. Demonstrate how to copy `.py` code into Jupyter notebook cells.
8. Discuss IBM Quantum account and API key safety.
9. Run or demonstrate the hardware template.
10. Assign line-by-line code review.

## Faculty teaching principle

Do not begin with abstraction alone. Begin with working code, visible output, and interpretation. Then ask students to modify one line at a time.

## API key safety

Faculty should repeatedly emphasize that API keys must never be committed to GitHub or included in screenshots.
