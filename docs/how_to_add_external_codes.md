# How to Add External Codes

This repository may include original code and freely available external code, but external code must be handled carefully.

Before adding external code:

1. Check the license.
2. Record the source URL.
3. Record the author or organization.
4. State whether the code is copied, adapted, or rewritten.
5. Add a metadata card.
6. Add a short explanation.
7. Do not remove original attribution.

If the license is unclear, do not copy the code directly. Link to the source and write a new teaching example.
