# Intermediate and Advanced Code Expansion

This update adds a broader set of code examples so that the library covers more of the intended intermediate and advanced roadmap.

## Intermediate coverage added

- QAOA for a simple MaxCut problem
- Noisy simulator comparison for a Bell state
- IBM hardware comparison template
- Phase estimation introduction
- Amplitude/probability estimation introduction

## Advanced coverage added

- H2-style quantum chemistry Hamiltonian with a VQE-like grid search
- Transverse-field Ising spin-chain Hamiltonian
- Heisenberg spin-chain Hamiltonian
- Simplified Hubbard-like Hamiltonian
- Quantum machine-learning kernel introduction
- Readout-error mitigation introduction
- OpenQASM runner notes

## Teaching caution

Some files are production-ready beginner scripts, while others are conceptual advanced teaching examples. Advanced files should be studied line by line and tested in the active Qiskit environment before being used in workshops.
