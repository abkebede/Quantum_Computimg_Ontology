Quantum Computing Applications Map
This guide connects code examples to natural problem types.
Problem type	Classical framing	Quantum-computing connection	Starter files
Probability and sampling	Random trials, distributions	Measurement counts, Sampler	`hadamard_superposition.py`, `sampler_examples.py`
Linear algebra	Vectors, matrices, eigenvalues	Statevectors, gates, observables	`estimator_zz_expectation.py`
Fourier analysis	Frequency decomposition	Quantum Fourier transform	`qft_intro.py`
Search	Find marked item	Grover search	`grover_intro.py`
Optimization	Minimize objective	QAOA, VQE	`real_amplitudes_ansatz.py`, `vqe_intro.ipynb`
Quantum physics	Hamiltonians and observables	SparsePauliOp, Estimator	`estimator_zz_expectation.py`
Chemistry/materials	Ground-state energy	VQE and simulation	`vqe_intro.ipynb`
Binary arithmetic	Logic and adders	Reversible gates, OpenQASM	`cuccaro_ripple_carry_adder.qasm`
The learning goal is to move from running short codes to understanding which natural problems motivate the code.
