# Code Source and License Policy

This library is intended for public GitHub collaboration. Therefore:

- Original examples are preferred.
- Adapted examples must cite source and license.
- External examples must be compatible with reuse.
- IBM Quantum API keys or tokens must never be committed.
- Screenshots must not show private credentials.

Recommended source labels:

- `original` — written for this repository.
- `adapted` — modified from an openly licensed example.
- `external` — copied only when license permits.
- `reference_only` — not copied; link and discussion only.
