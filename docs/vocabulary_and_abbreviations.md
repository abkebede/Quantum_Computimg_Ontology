# Vocabulary and Abbreviations

This glossary defines terms used in the Quantum Computing Code Study Library.

| Term | Meaning |
|---|---|
| QC | Quantum Computing. |
| QGSS | Qiskit Global Summer School or Qiskit training context. |
| `qgss_env` | Python virtual environment used for Qiskit work. |
| Qiskit | Python framework for quantum circuits, simulation, and IBM Quantum access. |
| Jupyter Notebook | Browser-based environment for running code in cells. |
| `.py` | Regular Python script. |
| `.ipynb` | Jupyter notebook file. |
| OpenQASM | Quantum assembly language for circuit descriptions. |
| Qubit | Quantum bit. |
| Classical bit | Ordinary bit used to store measurement output. |
| Gate | Operation applied to qubits. |
| Circuit | Sequence of gates and measurements. |
| Measurement | Reading qubits into classical bits. |
| Shot | One repeated circuit run. |
| Counts | Measurement outcome frequencies. |
| Sampler | Primitive for measurement distributions/counts. |
| Estimator | Primitive for expectation values. |
| Observable | Quantity measured from a quantum state. |
| Hamiltonian | Energy operator, often represented as a sum of Pauli terms. |
| Ansatz | Parameterized trial circuit. |
| VQE | Variational Quantum Eigensolver. |
| QFT | Quantum Fourier Transform. |
| API key/token | Private credential for IBM Quantum access; never commit it to GitHub. |
| Backend | Simulator or real quantum computer. |
| Transpilation | Rewriting a circuit for a target backend. |
| Ontology | Structured classification of codes, concepts, syntax, and applications. |
