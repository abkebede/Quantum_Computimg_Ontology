# Physics Literature Derived QC Documentation Policy

This activity is separate from the beginner Qiskit documentation.

## Local research track

Local research packages may include detailed equation extraction, table notes, figure interpretation, and paper-specific methodology notes when used privately for study.

## GitHub-safe track

Public GitHub versions should include:

1. citation,
2. short original summary,
3. independently written code,
4. no uploaded copyrighted PDFs,
5. no reproduced tables unless license permits,
6. no long copied passages.

## Required structure

Each literature-derived QC module should identify:

- physical theory,
- classical model,
- QC model,
- Hamiltonian representation,
- circuit representation,
- speed/scaling comparison,
- limitations,
- next research steps.
