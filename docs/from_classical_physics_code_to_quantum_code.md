# From Classical Physics Code to Quantum Code

In classical physics and mathematical physics, students learn to code:

- differential equations;
- matrix diagonalization;
- Fourier transforms;
- Monte Carlo sampling;
- optimization;
- eigenvalue problems;
- wave and quantum models.

Quantum computing connects to many of these same structures, but through quantum states, gates, observables, Hamiltonians, and measurement.

| Classical skill | Quantum computing pathway |
|---|---|
| Matrix multiplication | quantum gates and circuits |
| Eigenvalue problems | phase estimation, VQE |
| Fourier analysis | QFT |
| Optimization | QAOA, VQE |
| Monte Carlo | sampling and amplitude estimation |
| Hamiltonian mechanics/quantum mechanics | Hamiltonian simulation |

A code example should therefore be studied in two ways: as syntax and as a model of a problem class.
