
# Jupyter Visual Learning Path: Ginzburg–Landau Superconductivity

## Stage 1: Classical visualization
Open `gl_classical_visuals.ipynb`.

Study:
- how the order parameter varies in space
- how the free-energy landscape changes when the sign of `alpha` changes
- how superconducting order emerges below the transition

## Stage 2: QC visualization
Open `gl_qc_visuals.ipynb`.

Study:
- how a simplified superconducting-order Hamiltonian is represented in Qiskit
- what a pairing-inspired ansatz circuit looks like
- how a VQE-style energy curve behaves
- how runtime comparison should be interpreted honestly

## Stage 3: Interpretation
Write down:
- what the classical plots teach
- what the circuits teach
- what the VQE plot teaches
- why simulator timing is not the same as future quantum advantage
