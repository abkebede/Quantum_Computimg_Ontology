# Code of Conduct

This project is intended as a respectful educational resource for beginners, novice learners, faculty, and contributors.

Participants should:

- use respectful language;
- welcome beginner questions;
- explain corrections constructively;
- cite sources and licenses carefully;
- avoid uploading private credentials or sensitive information.

Unacceptable behavior includes harassment, insults, plagiarism, or knowingly contributing unsafe credential-handling examples.
