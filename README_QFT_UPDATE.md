## Purpose

This update adds a teaching unit for the Quantum Fourier Transform example. The unit explains the QFT code line by line, connects the circuit to Fourier structure and phase information, and places QFT in the larger pathway toward phase estimation, period finding, signal analysis, and advanced quantum algorithms.
